# ACIATII - sitio web

Sitio estático listo para publicar en Cloudflare Pages.

## Publicación rápida

1. Crea un repositorio en GitHub.
2. Sube esta carpeta.
3. En Cloudflare Pages, conecta el repositorio.
4. Framework preset: None.
5. Build command: dejar vacío.
6. Output directory: `/`.
7. Agrega tu dominio personalizado desde Cloudflare Pages.

## Editar contenido

- Página principal: `index.html`
- Estilos: `assets/styles.css`
- Inglés: `en/index.html`
- Alemán: `de/index.html`
